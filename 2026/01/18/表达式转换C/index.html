7-1 表达式转换

分数 25

作者 DS课程组

单位 浙江大学

算术表达式有前缀表示法、中缀表示法和后缀表示法等形式。日常使用的算术表达式是采用中缀表示法，即二元运算符位于两个运算数中间。请设计程序将中缀表达式转换为后缀表达式。

输入格式:

输入在一行中给出不含空格的中缀表达式，可包含+、-、*、/以及左右括号()，表达式不超过20个字符。

输出格式:

在一行中输出转换后的后缀表达式，要求不同对象（运算数、运算符号）之间以空格分隔，但结尾不得有多余空格。

这题卡的原因是因为存在负数和小数, 在判断数字的时候, 要同时考虑小数和负数, 例如2+(-2)这种情况, 判断为数字的条件, 需要改为if((s[i] == 0 || s[i - 1] == '(') && (s[i] == '+' || s[i] == '-') || s[i] == '. || isdigit(s[i])'

(s[i] == 0 || s[i - 1] == '(') && (s[i] == '+' || s[i] == '-')这段的逻辑就是在判断是否为(+2) 或者 (-2)这种

输出正数时, 如果是(+2)这种, 直接输出2即可

中缀转后缀, 遍历字符串,在遇见字符时,

如果是’(‘直接压进字符栈中,

如果是’)’, 循环输出栈顶元素直到遇到’(‘,

如果是+-*/, 需要判断优先级, 如果优先级 大于等于 栈顶元素, 输出栈顶元素, 直到优先级小于栈顶元素或者栈空

其次是注意格式的问题, 有可能存在只有一个数字的情况, 这时候就要在判断数字的时候做处理
