Hexo 快速入门：搭建属于自己的博客（从 0 到上线）

Hexo 是一个基于 Node.js 的静态博客框架：你写 Markdown，它帮你生成静态网页，然后把这些网页部署到 GitHub Pages、Vercel、Netlify 或服务器上即可访问。优点是：快、轻、可定制、维护成本低。

你将完成什么

  • 本地初始化 Hexo 博客
  • 选择并配置主题
  • 写第一篇文章
  • 部署上线（以 GitHub Pages 为例）
  • 常见问题与优化建议
